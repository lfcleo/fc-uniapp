## 介绍（Introduce）
FC-Uniapp 基于uni-app的快速开发框架。使用`vite`+`vue3`+`typescript`+`uniapp`+`pinia`+`unocss`技术栈。

包含以下功能

- 全局状态管理（Pinia）
- 路由管理（Vue Router）
- 国际化工具（i18n）
- 应用更新管理（Updated）
- 工具类（Utils）
- 网络请求工具（Request）
- 文件上传工具（Upload）
- 文件下载工具（Download）
- WebSocket工具（WebSocket）
- 系统权限管理（Authorize）
- 极光推送功能（JGPush）

等等

## 安装教程（Install）

以MacOS为例，（Windows和Linux系统请根据实际情况替换命令）

全局安装脚手架

```bash
sudo npm i -g fc-uniapp
```

然后在指定目录，执行以下命令创建项目

```bash
fc-uniapp
```

根据指示输入项目名称、所需功能模块。等待项目创建完成后执行

```bash
# 进入项目目录
cd <项目名称>

# 安装依赖
npm i
```

## 运行项目（Run）

使用HBuilderX运行项目

也可使用uni-app命令行工具运行项目。
